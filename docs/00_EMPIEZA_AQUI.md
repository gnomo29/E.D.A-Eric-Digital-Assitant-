# 00 - Empieza Aqui (sin saber programar)

Esta guia es para iniciar E.D.A. desde cero, paso a paso, sin tocar codigo.

## 1) Que es este proyecto

E.D.A. es un asistente de escritorio que:

- responde por texto y voz,
- abre/cierra apps,
- maneja recordatorios,
- usa memoria persistente,
- integra Spotify, YouTube y control remoto (si configuras credenciales).

## 2) Como arrancarlo en Windows

1. Abre la carpeta del proyecto.
2. Haz doble clic en `INICIAR_ASISTENTE.bat`.
3. Espera a que termine la instalacion (solo la primera vez tarda mas).
4. Se abre la interfaz del asistente.

## 3) Si algo falla al iniciar

- Verifica que tengas Python instalado.
- Repite `INICIAR_ASISTENTE.bat` una vez mas.
- Revisa `data/logs/installer.log`.
- Ejecuta `python tools/system_check.py` para ver estado rapido.

## 4) Primeros comandos para probar

- `abre chrome`
- `reproduce musica de queen`
- `busca videos de python`
- `recuérdame en 10 minutos tomar agua`
- `listar recordatorios`
